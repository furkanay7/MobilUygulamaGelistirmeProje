# MobilUygulamaGelistirmeProje

Bu projeyi çalıştırabilmek için bilgisayarınızda Node.js ve telefonunuzda Expo Go uygulaması yüklü olmalıdır.

1. Projeyi klonlayın veya indirin.
2. Proje dizininde terminali açın ve "npm install" komutunu çalıştırın.
3. "npm start" komutu ile uygulamayı başlatın.
4. Terminalde çıkan QR kodu okutun ve daha sonra telefonda uygulama açılacaktır.
